# meteocool

Makefile targets:

 - ```make run```: PNG + tiles erstellen in /tmp
 - ```make download```: aktuelle Wettermap vom DWD holen nach /tmp


## Install

# Mac OS

```brew install gdal --HEAD && brew install hdf5 netcdf4 && pip3 install wradlib gdal geojson```

# Linux

```apt-get install gdal-dev && ???????```
